# dse: Dutch Soccer Exchange

![Node.js CI](https://github.com/ejjaquet/dse/actions/workflows/node.js.yml/badge.svg)


A Vue3 project setup with,

-   Vite
-   Router (*)
-   Vuex (*)
-   Eslint (*)
-   Prettier (*)
-   Tailwind CSS
-   and some custom preferences.

### Run this project:

```
npm
npm run dev
```
